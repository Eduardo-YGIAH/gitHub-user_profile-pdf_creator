# PDF-Creator from GitHub profiles

This is an example project of a PDF creator in Node.js.
User is able to insert a GitHub username and a prefered color and the software will generate a pdf using the GitHub API and packages like puppeteer and handlebars.
Please see package.json file for full list of dependencies.

## Forking and cloning this project

If you would like to run this code in your computer, be sure to change the filename .env_sample to .env and insert the full GitHub url with your API Key.
